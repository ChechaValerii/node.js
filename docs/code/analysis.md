# Analysis

Tool for identifying problematic patterns found in JavaScript code.

- [ESLint](https://eslint.org/)
- [Sonar](https://www.sonarlint.org/)

## ESLint

### Installation

```bash
npm install eslint --save-dev
# or
yarn add eslint --dev
```

### Initialization

You should then set up a configuration file:

```bash
npx eslint --init
```
